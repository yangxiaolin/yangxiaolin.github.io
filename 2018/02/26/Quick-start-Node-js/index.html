Quick start Node.js

Quick start

大家对Node.js有很多称呼，NODE.JS、nodejs等等。我参考Node.js官网的叫法，以下全文称呼为Node.js。

Node.js是什么？和JavaScript有什么关系？网上有很多答案，本文就不再赘述了。引用Node.js官网的话–Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine.

Node.js安装：

  1. Mac、Windows 可以下载对应的安装包，按部就班的安装就可以了。Linux系统我也还没尝试过，可以参考Node.js官网下载源码编译安装。

  2. 安装完成后，打开最顺手的终端。输入 node -v 查看是否安装成功。如下图：


  当输出了安装的Node.js的版本后，就证明安装成功了，恭喜。如果没有安装成功，没关系，网上教程一抓一大把。(嘻嘻？？？)
