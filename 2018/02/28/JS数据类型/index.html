JS数据类型

JS数据类型

  • Primitive value (String, Number, Boolean, Undefined, Null, Symbol) -> 基本数据类型
  • Reference value (Object) -> 引用类型

存储位置

  • Primitive value
    • 存储在栈内存中
    • 栈内存里包括了变量的标识符和变量的值
    • 在栈内存里大小固定
  • Reference value
    • 存储在栈内存和堆内存里
    • 栈内存保存变量标识符和指向堆内存中该对象的指针
    • 堆内存里保存了变量的值
